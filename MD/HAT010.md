# HAT-010 5x5 매트릭스

## 인스턴스 선언

HAT-010 5x5 매트릭스(HAT010) 블록을 작업 영역에 추가하면, Python 코드에는 다음과 같은 인스턴스 선언이 자동으로 삽입됩니다:

```python
cheesestick = CheeseStick(0)
hat010 = cheesestick.HAT010()
```

### 매개변수

| 이름 | 구분 | 설명 | 범위 / 종류 | 기본값 |
| --- | --- | --- | --- | --- |
| index | 드롭다운 옵션 | 인스턴스 번호 (0부터 시작) | 0 이상 정수 | 0 |


<br>

| 이름 | 구분 | 설명 | 범위 / 종류 | 기본값 |

## 매트릭스 이동하기

지정한 x, y만큼 화면을 이동합니다.  
x 값이 양수이면 화면이 오른쪽으로 이동하고, 음수이면 왼쪽으로 이동합니다.  
y 값이 양수이면 화면이 아래로 이동하고, 음수이면 위로 이동합니다.

<img src="https://raw.githubusercontent.com/RobomationLAB/User_Guide_KR/main/assets/cheesestick_ext/HAT010/6.png" alt="매트릭스 이동하기">

### 매개변수

| 이름 | 구분 | 설명 | 범위 / 종류 | 기본값 |
| --- | --- | --- | --- | --- |
| x | 입력값 (필드) | 가로 이동량 | 정수 | - |
| y | 입력값 (필드) | 세로 이동량 | 정수 | - |

<!-- 
### JavaScript
```javascript
$('CheeseStick*0:HAT>HAT010+origin.x').d = $('CheeseStick*0:HAT>HAT010+origin.x').d + 1;
$('CheeseStick*0:HAT>HAT010+origin.y').d = $('CheeseStick*0:HAT>HAT010+origin.y').d + 0;
```
-->

### Python
```python
cheesestick = CheeseStick(0)
hat010 = cheesestick.HAT010()

hat010.shift(1, 0)
```
